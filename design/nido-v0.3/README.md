# Nido v0.3 — baseline visual canónico

La única fuente visual vigente dentro del HTML generado es la sección estructural `t3`, correspondiente a Nido v0.3. Sus tokens, componentes, navegación, responsive y comportamiento de interfaz son la referencia para implementación.

Las menciones a v0.1 o v0.2 que aparecen dentro de changelogs, etiquetas de origen o anotaciones del HTML son historial del proceso de diseño. No representan baselines alternativos y no deben implementarse.

## Archivos

- `Nido - Sistema y pantallas.dc.html`: referencia visual generada. No es código de producción.
- `support.js`: soporte del documento generado; tampoco es código de aplicación.
- `README.md`: alcance y procedencia del baseline.

No se debe copiar el runtime, JavaScript generado ni los estilos desorganizados del HTML. `docs/system-design.md` prevalece para dominio, cálculos, seguridad y alcance; la sección `t3` prevalece para presentación y comportamiento visual.
